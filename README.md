# 🧪 课堂小探究 \- Venn 图互动演示课堂小探究 \- Venn 图互动演示

 基于 HTML \+ SVG \+ JavaScript 实现的集合运算交互式韦恩图演示工具，适用于数学课堂教学、知识点可视化讲解

__ ✨ 项目简介__

  本项目是一个纯前端实现的交互式韦恩图（Venn Diagram），通过可视化方式展示集合运算（对立事件、交集、分离等），界面简洁美观、交互流畅，无需后端服务，直接在浏览器中运行。

__ 🚀 功能特性__

 ✅ 对立事件高亮：一键展示集合 B 的对立事件

 ✅ 区域筛选：只保留 A 中 B 的对立事件区域

 ✅ 交集标注：清晰高亮 A、B 集合的交集部分

 ✅ 集合淡出：可隐藏集合 B，聚焦单一集合

 ✅ 交集分离：动态分离交集区域，强化视觉效果

 ✅ 一键重置：快速恢复初始状态，重复演示

__🎨 界面特点__

现代化卡片式设计，毛玻璃质感背景

渐变色按钮与平滑过渡动画

响应式布局，适配电脑 / 平板 / 手机

高对比度配色，教学演示清晰易懂

__📦 使用方法__

1\.直接运行：将代码保存为 \.html 文件，用任意浏览器打开即可使用

2\.课堂演示：全屏展示，通过按钮控制演示节奏

3\.二次开发：纯原生 HTML/CSS/JS，无框架依赖，轻松修改扩展
