## 🧪 子项目一：独立事件分析器

### 功能简介

模拟**有放回地从袋中取球**（数字 1~6），随机生成或手动输入两个事件，自动计算并判断两事件是否相互独立。

### 核心特性

| 特性 | 说明 |
|---|---|
| **双模式** | 自动模式（随机生成事件对）/ 手动模式（自由输入事件描述） |
| **智能解析** | 支持四种事件类型：指定第一次取值、指定第二次取值、指定两数和、指定两数积 |
| **独立判定** | 基于古典概型（36 种等可能样本），比较 P(AB) 与 P(A)·P(B) |
| **友好 UI** | 占位符提示、错误捕获、结果颜色反馈（绿色 ✅ / 红色 ❌） |

### 事件描述格式

| 类型 | 格式示例 |
|---|---|
| 第一次取球 | `第一次取出的球的数字是3` |
| 第二次取球 | `第二次取出的球的数字是5` |
| 两数之和 | `两次取出的球的数字之和是7` |
| 两数之积 | `两次取出的球的数字之积是12` |

### 运行方式

```bash
# 需要 Python 3 + Tkinter（标准库自带）
python 666.py
```

### 技术栈

- **Python 3** + **tkinter** / **ttk**
- 自定义 `PlaceholderEntry` 控件（带占位符的输入框）
- 随机数生成 + 列表推导式枚举样本空间

### 数学原理

样本空间 Ω = {(i, j) | i, j ∈ {1, 2, 3, 4, 5, 6}}，共 36 个等可能基本事件。

判定逻辑：

```
P(A) = |A| / 36
P(B) = |B| / 36
P(AB) = |A ∩ B| / 36

若 |P(AB) - P(A)·P(B)| < 1e-9，则 A、B 相互独立
```

> 💡 自动模式下，程序会智能生成**独立**或**不独立**的事件对，供学生探究发现规律。

## 🎨 子项目二：Venn 图互动演示

### 功能简介

一个轻量级的 **集合运算 Venn 图可视化工具**，通过 SVG 动画演示集合的交、并、补及分离操作，适用于课堂探究式教学。

### 交互按钮说明

| 按钮 | 功能 |
|---|---|
| 🟣 表示 B 的对立事件 | 高亮显示 B 的补集（A 中也有视觉反馈） |
| 🟢 只保留 A 中的 B 的对立事件 | 显示 A ∩ B̄（即 A 中不属于 B 的部分） |
| 🟡 表示 AB | 高亮 A ∩ B 的交集区域 |
| 🟣 淡出集合 B | 隐藏集合 B，聚焦集合 A |
| 🩷 分离 | 将交集区域从 B 中分离出来，直观展示 AB |
| ⚫ 重置 | 恢复初始状态 |

### 技术亮点

- **纯前端实现**：HTML + CSS + SVG + Vanilla JavaScript，无需任何依赖
- **SVG 路径运算**：通过几何计算精确求解两圆交集的弓形路径
- **CSS 动画**：`transition` + `transform` 实现平滑的分离/淡入淡出效果
- **fill-rule: evenodd**：巧妙利用 SVG 填充规则实现"挖洞"效果（如补集）

### 运行方式

直接用浏览器打开即可：

```bash
# 或用任意 HTTP 服务器
python -m http.server 8080
# 访问 http://localhost:8080/venn.html
```

### 数学对应

| 操作 | 集合表达式 |
|---|---|
| 表示 B 的对立事件 | B̄ = U \ B |
| 只保留 A 中的 B 的对立事件 | A ∩ B̄ |
| 表示 AB | A ∩ B |
| 分离 | 将 A ∩ B 从 B 中视觉分离 |

## 🎯 教学应用场景

1. **概率独立性探究**：用 `666.py` 生成大量事件对，引导学生归纳独立事件的判定条件
2. **集合运算可视化**：用 `venn.html` 在课堂上动态演示 Venn 图的各类运算
3. **两者结合**：先用 Venn 图理解事件关系，再用分析器验证概率计算
